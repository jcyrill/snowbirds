# Drafts Folder

## Version Control Guidelines

### Major Versions (X.0)
- **v0.1** - Outline/Treatment
- **v1.0** - First complete draft
- **v2.0** - Major revision (significant structural changes)
- **v3.0** - Second major revision
- **v4.0** - Polish draft

### Minor Versions (X.Y)
- **v1.1, v1.2, etc.** - Small revisions, scene tweaks, dialogue polish within same draft

### File Naming
```
snowbirds_v1.0_2025-11-30.fountain
snowbirds_v1.1_2025-12-05.fountain
snowbirds_v2.0_2026-01-15.fountain
```

## Screenplay Format Options

### Fountain (.fountain)
**Pros:**
- Plain text format
- Works in any text editor
- Version control friendly (git)
- Free
- Converts to PDF easily

**Cons:**
- Less visual than dedicated software
- Need to learn syntax

**Recommended tools:**
- Highland 2 (Mac)
- Fade In
- WriterSolo
- VS Code with Fountain extension

### Final Draft (.fdx)
**Pros:**
- Industry standard
- Rich formatting tools
- Revision tracking
- Collaboration features

**Cons:**
- Expensive ($249)
- Proprietary format

### Fade In (.fadein)
**Pros:**
- Final Draft alternative ($79)
- Compatible with Final Draft
- Great features

**Cons:**
- Still costs money
