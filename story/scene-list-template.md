# Scene List - Snow Birds

## How to Use This Template

For each scene, fill in:
- **Scene #** - Sequential numbering
- **Location** - Where it takes place
- **Time** - Day/night, time of day
- **Characters** - Who's in it
- **Purpose** - What this scene accomplishes (plot, character, comedy, etc.)
- **Key Beats** - Main story/emotional moments
- **Comedy** - Funny moments, jokes, set pieces
- **Page Estimate** - Rough page count
