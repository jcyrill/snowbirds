# Reference Materials

## Folder Contents

This folder is for visual and audio reference materials:

### Images
- **Character inspiration** - photos of people who inspire your characters
- **Location photos** - 90s interiors, exteriors, fashion
- **Fashion reference** - 90s style examples
- **Mood boards** - visual tone/style collection

### Music
- Create playlists by year (1990-1999)
- Note specific songs for specific scenes
- Soundtrack ideas

### Video
- Links to reference films/TV episodes
- Clips that inspire tone or style
